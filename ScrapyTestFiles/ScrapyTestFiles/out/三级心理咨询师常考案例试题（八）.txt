	案例八： 一般资料：张某，男性，23 岁，某公司员工.     	同事反映：张某在举止上有明显改变，常常自言自语，有时傻笑，有时烦躁不安或气愤不己。原先工作表现良好，要强、上进，现在和以前完全不一样了。张某学习回来后，公司同事米听说哪位女士喜欢他，也没有听说有人议论。 	多选：1 、张某的行为症状包括( )。 	( A )有时傻笑     ( B )自言自语 	( C )烦躁不安     ( D )气愤不已  	多选：2 、张某的情绪症状包括( )。 	( A )烦躁       ( B )困惑 	( C )要强       ( D )愤怒  	多选：3 、案例中反映出张某原来的人格特征包括( )   	( A )要强         ( B )古怪 	( C )上进         ( D )执著  	单选：4 、张某的人格变化表现在( )。 	( A )心情烦躁       ( B )举止异常 	( C )气愤不己       ( D )主动求医  	多选：5 、张某存在主观和客观不一致，具体表现包括( )。	( A )百思不得其解 ( B )坚信某女爱慕自己 	( C )常常自言自语 ( D )觉得同事议论自己  	单选：6、张某的自知力完整程度是( )。 	( A )保持完整           ( B )完全受损 	( C )部分受损           ( D )无法确定  	( A )自知力受损           ( B )主动求医 	( C )自知力完整           ( D )被动求医  	单选：8 、张某最主要的症状表现是( ) o 	( A )感知障碍              ( B )情感障碍 	( C )思维障碍              (O )意志障碍  	多选：9、张某具体症状包括( )。 	( A )被害妄想        ( B )钟情妄想                            	( C )关系妄想        ( D )疑病妄想 	单选：10 、对张某最可能的初步诊断是 	( A )一般心理问题       ( B )严重心理问题                   	( C )神经症性问题       ( D )精神病性问题

		案例介绍：外出学习三个月归来后，发现同事们对自己的态度不对，都在背后议论自己，也不知为什么。经过一段时间的观察发现，原来是和自己一起学习的另一家公司的一位年轻女士对自己有爱慕之心，不知道通过什么高科技手段.把她的爱意告诉了自己单位的同事，所以大家才议论纷纷。可是自己百思不得其解，为什么她不直接告诉自己，而搞得满城风雨，为此前来寻求答案。     		单选：7 、张某对于那位女士不直接向自己表达爱意，而搞得满城风雨的做法不理解，于是找心理咨询师寻求答案，这表明( )。
