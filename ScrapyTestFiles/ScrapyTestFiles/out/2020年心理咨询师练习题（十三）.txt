2020年心理咨询师练习题（十三）
	一般资料：赵某，女，48岁，机关退休干部。	案例介绍：赵某大学毕业后进入某机关，个人素质较高，工作能力较强，但人际关系较差，常为一些小事和同事闹得不欢而散，生活中几乎没有知心朋友。还怀疑丈夫有外遇，夫妻关系紧张，看见别人说话，总认为别人说自己坏话，单位里有人要整自己，内心很紧张，不断向上级机关反映。去年单位人员调整，领导让她退休，她想不开，寻死觅活地说是有人故意陷害她，经常站在窗口笑笑哭哭，手舞足蹈。最近半年总说听见有人敲家里的门。平常散步时，看见邻居和孩子说说笑笑，就说她们是在讲自己的坏话，要丈夫和孩子提高警惕，不要让坏人钻了空子。	心理咨询师观察了解到的情况：赵某从小生活条件优越，成长较为顺利，学习成绩良好。在单位能胜任工作。单位人员反映，没听说有人背后讲她的坏话。丈夫反映赵某原来较勤快，但近来赵某不做家务，不与父母联系，不关心女儿。丈夫多次要她去看医生，她都说自己没病，有病的是单位领导。在丈夫逼迫下，赵某勉强前来咨询。	多选：赵某的症状包括( 	)	A.错觉	B.妄想	C.幻觉	D.紧张	单选：对赵某近期社会性功能损害程度的评价是( 	)	A.功能没有受损	B.轻一中度损害	C.功能完全丧失	D.中一重度损害	多选：赵某认为别人讲自己的坏话，属于( 	)	A.自罪妄想	B.被害妄想	C.夸大妄想	D.关系妄想	单选：从案例判断赵某说有人故意陷害她，属于( 	)	A.被害妄想	B.客观现实	C.自罪妄想	D.目前无法判断	【解析】需要深入了解是否有人故意陷害。	单选：从本案例判断赵某的病程为( 	)	A.数月	B.一年	C.半年	D.多年	单选：赵某认为自己没病反映的是( 	)	A.无判断力	B.无理解力	C.无自知力	D.无领悟力	单选：心理咨询师掌握赵某心理异常症状，是为了( 	)	A.对变态人格进行有效的咨询	B.诊断精神障碍和进行转介	C.对精神病患者进行心理咨询	D.诊断精神障碍和进行治疗	【解析】咨询师需诊断求助者是否为精神障碍，如果是需要转介相关机构。	多选：心理咨询师在判断此类问题时，须掌握的重要原则是( 	)	A.主客观的统一性	B.心理活动表现稳定性	C.个性相对稳定性	D.精神活动内在一致性	【解析】判断求助者心理问题性质的原则：主客观统一性、人格是否相对稳定、心理活动是否一致。	单选：赵某的心理健康水平属于( 	)	A.心理正常	B.心理不正常	C.心理健康	D.心理不健康	单选：对赵某的诊断最可能的是( 	)	A.一般心理问题	B.神经症	C.严重心理问题	D.精神障碍	【解析】求助者无自知力，并有妄想、幻听等症状，可能是精神障碍。	多选：会谈目标中若有一个以上的内容，心理咨询师应该( 	)	A.分清主次	B.摄入性会谈	C.同时处理	D.连珠式提问	多选：心理咨询师对赵某妥当的处理是( 	)	A.尝试进行心理咨询	B.请精神科医生会诊	C.尝试进行心理治疗	D.转介至精神病医院	【解析】求助者出现妄想和幻听症状，应请心理医生会诊，确认是否属于精神病。如确诊应转精神病院，因为精神病不属于心理咨询的范围。	【分析说明】	症状：总认为别人说自己坏话、有人整自己、内心紧张、经常站窗口说说笑笑、总说听见有人敲家门、怀疑丈夫有外遇，这些表明求助者可能有妄想和幻觉，并情绪紧张。	社会功能受损：不做家务、不与父母联系、不关心女儿，社会功能严重受损。	求治动机：在丈夫的逼迫下前来咨询，属于被动求治。	自知力：说自己没病，表明无自知力。
